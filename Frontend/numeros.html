Números no JavaScript

Números podem ser escritos com ou sem vírgula

Números muito grandes e muito pequenos podem ser escritos em notação científica.

Números inteiros (escritos sem vírgula ou ponto decimal e sem expoente) são precisos até 15 dígitos.

Aritmética de ponto flutuante nem sempre é 100% exata;

Se você adicionar dois números, o resultado será um número.

Se você adicionar duas STRINGs numéricas, o resultado será uma string concatenada.

Se você adicionar um número e uma string, o resultado será uma string concatenada.

Um erro comum é esperar que o resultado de "O resultado é" + x + y seja 30.

Um erro comum é esperar que o resultado abaixo seja 102030

O JavaScript sempre tentará converter strings em números em todas as outras operações numéricas.
