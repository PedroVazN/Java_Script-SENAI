Métodos para Números em JavaScript

O método toString() pode exibir números em base 2 até 36.

O método toExponential() retorna uma string com o número arredondado e escrito usando a notação exponencial.

O método toFixed() retorna uma string com o número escrito com uma quantidade especificada de casas depois da vírgula. toFixed(2) é usando especialmente com dados monetários.

O método Number() converte valores em números.

O método parseInt() analisa uma string e retorna um número inteiro. Espaços são permitidos. Somente o primeiro número é retornado.

O método parseFloat() converte strings em números de ponto flutuante
